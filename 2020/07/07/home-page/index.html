Projects

  • Current (Extra-curricular Course) Project: we used transfer learning with a pretrained CNN (MobileNet-V2) on Market1501 dataset, initially getting accuracy of 50%. After blending label smoothing, the accuracy increases to 55%. Finally, introducing semi-hard triplet loss improves the accuracy to 75%.
  • July 2018 at MIT (11th Grade): we collect data using web crawler. Then we use SURF to extract features and BOVW model to save features for feature matching. The final robot was able to recognize 70% physical bottles.
