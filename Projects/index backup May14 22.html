Projects

  • Anime Segmentation: originally COGS118B final project Slides, further improving the model performance. Use CartoonGAN on ADE20K dataset to get the Anime-style images with the same segmentation mask, then apply DeepLabV3 to train on transferred images.
  • (Extra-curricular Course) Project —— Person-ReID: we used transfer learning with a pretrained CNN (MobileNet-V2) on Market1501 dataset, initially getting accuracy of 50%. After blending label smoothing, the accuracy increases to 53%. Finally, introducing semi-hard triplet loss and replacing a more robust backbone improve the accuracy to 73%.
  • July 2018 at MIT (11th Grade): we collect data using web crawler. Then we use SURF to extract features and BOVW model to save features for feature matching. The final robot was able to recognize 70% physical bottles.
